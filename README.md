# Vehicle Rental System - Database Design & SQL Queries

## Project Overview
This project demonstrates database design and SQL query implementation for a Vehicle Rental System. It covers ERD design, table relationships, and SQL queries using JOIN, EXISTS, WHERE, GROUP BY and HAVING.

## Database Entities
- Users
- Vehicles
- Bookings

## Relationships
- One User can make multiple bookings
- One Vehicle can be booked multiple times
- Each booking is linked to exactly one user and one vehicle

## Concepts Used
- Primary & Foreign Keys
- One-to-Many Relationships
- INNER JOIN
- NOT EXISTS
- GROUP BY & HAVING
